# agent_sdlc

Lightweight package that extracts agent-assisted SDLC primitives: provider abstraction, retry utilities, and adapter-friendly agents.

Quick start

1. Create a virtualenv and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Run unit tests:

```bash
pytest -q
```

Purpose: this repository contains building blocks intended to be imported into other projects or published as a small supporting library.
