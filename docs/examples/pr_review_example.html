PR Review Example

This page documents the `pr_review_example.py` script.

What it does

Runs the PR review agent with a DummyLLMProvider to produce deterministic findings.

How to run

python agent_sdlc/examples/pr_review_example.py

Sample output

[
  {
    "title": "Use f-strings",
    "description": "Replace string concatenation with f-strings for readability",
    "severity": "low"
  }
]
