agent_sdlc — Examples

These examples demonstrate running the sample agents locally using the deterministic DummyLLMProvider.

  • PR Review Example — run the PR review agent on a sample diff.
  • Issue Refinement Example — refine an issue description into tasks.

Run locally

From the repository root:

python agent_sdlc/examples/pr_review_example.py
python agent_sdlc/examples/issue_refinement_example.py
