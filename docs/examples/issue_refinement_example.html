Issue Refinement Example

This page documents the `issue_refinement_example.py` script.

What it does

Runs the Issue Refinement agent with a `DummyLLMProvider` to generate proposed tasks and suggestions.

How to run

python agent_sdlc/examples/issue_refinement_example.py

Sample output

[
  {
    "title": "Break into subtasks",
    "description": "Split the feature into API, frontend, and tests",
    "severity": "info"
  }
]

See the script: issue_refinement_example.py
